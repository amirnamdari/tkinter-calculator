# Tkinter Calculator

This is a modern calculator app built with Python and Tkinter.

Features:  
- Clean and user-friendly interface  
- Supports basic math operations: addition (+), subtraction (-), multiplication (*), division (/)  
- Advanced functions: square root (√), exponentiation (^), logarithms (log, ln), and constants (π, e)  
- Works with both keyboard and mouse  
- No need to install Python to run (standalone exe version included)

---

To run the program, simply execute the `calculator.exe` file located inside the `dist` folder.
